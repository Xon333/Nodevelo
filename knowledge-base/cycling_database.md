# Cycling Database

Reference data for session construction. All power targets are % of FTP.

## Training zones (7-zone model)

| Zone | Name | % FTP | Typical use |
|---|---|---|---|
| Z1 | Active recovery | <55% | Recovery spins, between intervals |
| Z2 | Endurance | 56–75% | Aerobic base, fat oxidation, durability |
| Z3 | Tempo | 76–87% | Group ride pace; use sparingly in structured blocks |
| Z4 | Threshold | 88–105% | FTP development, sweet spot (88–94%) |
| Z5 | VO2max | 106–120% | Aerobic ceiling, 2–6 min intervals |
| Z6 | Anaerobic | 121–150% | 30 s – 2 min capacity work |
| Z7 | Sprint | max | Neuromuscular power, 5–15 s |

## Session templates

Durations scale with athlete availability. Power targets in Intervals.icu syntax.

### Z2 endurance

```
Warmup
- 10m ramp 45-65%

- 1h30m 65% 90rpm

Cooldown
- 10m 50%
```

Progression: extend total duration before raising intensity. Add 3x1m 105% "openers" mid-ride for variety without breaking the aerobic intent.

### Threshold 3x12 (entry) → 3x15 → 2x25 (advanced)

```
Warmup
- 15m ramp 50-75%

Main Set 3x
- 12m 95%
- 4m 55%

Cooldown
- 10m 50%
```

Progress time-at-threshold ~10% per week within a block: 36 → 40 → 45 min. Sweet spot variant: 88–92% for longer totals (50–70 min).

### VO2max 5x4

```
Warmup
- 15m ramp 50-75%
- 3x 30s 110% with 90s easy between

Main Set 5x
- 4m 112%
- 4m 50%

Cooldown
- 10m 50%
```

Target: maximal sustainable aerobic power for the interval length; breathing should be the limiter. 12–20 min total work time.

### SIT / sprint intervals

```
Warmup
- 15m ramp 50-75%

Main Set 6x
- 30s 150%
- 4m30s 45%

Cooldown
- 10m 50%
```

Full recoveries protect quality. Stop the set when power drops >10% below the first rep.

### Recovery spin

```
- 45m 45-55% 95rpm
```

Strictly capped intensity. Skipping it entirely is always acceptable.

### Durability sessions

Place key intervals at the END of a long Z2 ride (e.g. 2 h Z2, then 3x8m 92%) to train resistance to fatigue. Use only for athletes with an established base and in weeks 2+ of a block.

## Strength (off-bike)

45–60 min: squat or leg press, hip hinge, step-up/lunge, core. 3–4 sets of 4–8 reps at controlled heavy load in build blocks; maintain with 1 session/week during heavy bike weeks. Schedule on the same day as a hard ride (hard days hard) or on easy days at least 6 h apart from quality intervals — never the day before a key session.
